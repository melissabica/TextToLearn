Welcome to Text to Learn

Text to Learn is a training tool developed for social enterprises in emerging markets. It uses SMS to distribute training materials to users' mobile phones, and it provides social enterprises with a web-based dashboard to upload and send training materials, manage users, and create SMS-based quizzes to assess users' progress.

Navigating the Website

Training Materials

Upload and manage training materials here. Training materials consist of a title, tag (which is used by trainees to respond to texts), main training text, and optional quiz of up to five questions. Once a training material has been saved, it can be assigned to users from the main list, which will send them a notification text message.

Manage Users

Users can be added and edited here. All that is needed to register a user for the system is a name and phone number. When a new user is added, they will receive a confirmation text message.

Message Tester

This is a simulation of a phone interacting with the system. Use this when a mobile phone is not available, or to test training materials and quizzes.

Message Log

This is a record of every text message sent and received by the system. Messages can be filtered by user by clicking on their name in the list.
